# 🎬 Movie Recommendation System

This project is a **Movie Recommendation System** built using **Python, Pandas, Scikit-Learn, and Streamlit**.  
It recommends movies based on similarity using **cosine similarity**.

## 🚀 Features

- Recommends similar movies
- Interactive web interface
- Built with Streamlit
- Uses collaborative filtering technique

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Git & GitHub

## 📂 Project Structure

Movie-Recommendation-System
│
├── app.py
├── movie_recommendation.py
├── movies.csv
├── ratings.csv
└── README.md

## ⚙️ How It Works

1. Load movie and rating datasets
2. Merge the datasets
3. Create a user-movie matrix
4. Calculate cosine similarity between movies
5. Recommend top similar movies

## ▶️ How to Run the Project
Install required libraries:
pip install pandas scikit-learn streamlit
Run the Streamlit application:
streamlit run app.py
